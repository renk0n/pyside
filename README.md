# 画像解析・加工アプリ

PySide6とOpenCVを用いて、GUIで直感的に操作できる画像処理アプリを作成しました。

## 工夫した点・実装内容

ただ画像を表示して加工するだけでなく、実用的なツールとして使えるようにいくつか工夫を入れました。

UIの設計では、PySide6のレイアウト機能を使い、左側に操作パネル、右側にスクロール可能な画像エリアを配置して見やすくしています。機能面では、グレースケールや大津の二値化といった基本処理に加え、OpenCVのHaar Cascade分類器を使った顔認識機能も実装しました。

また、Canny法でのエッジ抽出はGUIの強みを活かし、スライダーを動かして閾値を動的に調整できるようにしています。操作していて不便にならないよう、「1つ前の状態に戻す（Undo）」機能や、日本語のフォルダ名でもエラーにならずに画像を読み書きできる処理も組み込みました。

## 動かし方

1. 仮想環境等で、必要なライブラリをインストールします。
   pip install -r requirements.txt

2. 以下のコマンドでアプリを起動してください。
   python main.py
